# Microsoft 365 Support Concepts

This file documents beginner Microsoft 365 support concepts for IT support practice.

## Purpose

The goal of this lab is to understand common Microsoft 365 tools and support issues that may come up in help desk, desktop support, and IT support roles.

## What Is Microsoft 365?

Microsoft 365 is a cloud-based productivity platform used by many businesses. It includes tools for email, file storage, collaboration, meetings, messaging, and office productivity.

Common Microsoft 365 tools include:

* Outlook
* Teams
* OneDrive
* SharePoint
* Word
* Excel
* PowerPoint
* Microsoft 365 admin concepts
* Microsoft Entra ID concepts

## Common User Support Issues

### Outlook Issues

Common Outlook support issues may include:

* User cannot send or receive email
* Outlook will not open
* Mailbox is full
* Email is stuck in outbox
* User cannot find an email
* Calendar invite issues
* Shared mailbox access issues

### Teams Issues

Common Teams support issues may include:

* User cannot join a meeting
* Microphone or camera is not working
* Audio issues
* User cannot access a team or channel
* Chat notifications are not working
* Teams needs to be updated or restarted

### OneDrive Issues

Common OneDrive support issues may include:

* Files are not syncing
* User cannot find synced files
* OneDrive is signed into the wrong account
* Sync errors
* Storage issues
* File conflict errors

### SharePoint Issues

Common SharePoint support issues may include:

* User cannot access a site
* Permission issues
* File sharing problems
* Missing files or folders
* Syncing SharePoint libraries to OneDrive

## Account and Access Concepts

Microsoft 365 support often involves account and access troubleshooting.

Common areas to check include:

* Username or email address
* Password reset needs
* MFA prompts
* Account lockouts
* Group membership
* License assignment
* Permission levels
* Whether the issue affects one user or multiple users

## Example Troubleshooting Scenario

### Problem

A user says they cannot access a SharePoint folder.

### Steps to Check

1. Confirm the user is signed into the correct Microsoft 365 account.
2. Confirm the exact SharePoint site or folder they are trying to access.
3. Ask whether they receive an error message.
4. Check whether other users can access the same folder.
5. Confirm whether the user has the correct permissions.
6. Check whether the issue is browser-related.
7. Try a different browser or private browsing window.
8. Document the issue, steps taken, and resolution or escalation.

## IT Support Documentation Notes

When documenting a Microsoft 365 issue, I should include:

* User affected
* Application affected
* Error message
* Device or browser used
* Whether the issue affects one user or multiple users
* Troubleshooting steps taken
* Access or permission details
* Final resolution or escalation notes
